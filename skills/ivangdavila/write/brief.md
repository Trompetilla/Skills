# Brief Reference

Consult when planning any writing task.

## Before Writing, Confirm:

| Question | Why |
|----------|-----|
| **Audience** | Expert vs beginner changes everything |
| **Purpose** | Inform, persuade, entertain, instruct? |
| **Tone** | Formal, casual, technical, warm? |
| **Length** | "Quick answer" or "comprehensive"? |
| **Platform** | Tweet vs email vs docs vs chat |

## When to Outline

- Piece longer than 300 words
- Multiple sections or arguments
- User said "article", "post", "guide", "report"

## Outline Format

```
1. Hook/Opening — why reader should care
2. Main point 1 — with example
3. Main point 2 — with example
4. Conclusion — action or takeaway
```

## Don't Plan

- Quick replies under 100 words
- Simple answers to direct questions
- Chat messages
